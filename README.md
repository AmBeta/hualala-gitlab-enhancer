# hualala-gitlab-enhancer

优化哗啦啦 gitlab 使用体验的油猴脚本。

## Features

- 浏览项目文件时左侧显示项目目录结构，可快速定位到文件
- 浏览 Markdown 文件时右侧显示文档目录，可快速定位到文档位置
- 更完美的 Markdown 文件预览支持
